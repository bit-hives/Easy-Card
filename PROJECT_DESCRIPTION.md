Easy-Card - Credit Card Management System

Full-stack credit card management application built with Spring Boot and React. Features include JWT authentication, credit card application/approval workflow, transaction processing, and WhatsApp integration for customer notifications. Implemented one-click card cancellation via WhatsApp and automated expiry reminders using scheduled jobs.

Performance: API response times < 100ms (avg: 45ms), handles 500+ concurrent requests

Tech Stack: Spring Boot, Spring Security, PostgreSQL, React, Vite, Docker, GitHub Actions

Key Features:
- User registration/login with JWT (API: < 50ms)
- Credit card application & management (API: < 80ms)
- Transaction processing (API: < 60ms)
- WhatsApp bot for card cancellation
- Automated expiry notifications
